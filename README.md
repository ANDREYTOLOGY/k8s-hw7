# Домашнее задание занятию `«Установка Kubernetes»` - `Чернышов Андрей`

В рамках выполнения задания выполнена демонстрация ранее развернутого `Kubernetes-кластер`, эксплуатируемой в рабочей инфраструктуре. Кластер был развернут с использованием `kubeadm`.  
Конфигурация кластера соответствует требованиям задания.  
в качестве `Container Runtime` используется `containerd`;  
сервис `etcd` размещён на `master-ноде`.
